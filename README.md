# Angelus Bartender Backend

Backend project for Angelus Bartender web page. Here are two micro-service build in [Go](https://golang.org), the ms-users and the ms-cocktails. Also you cand find a postman collectiosn with all endpoins and docker-compose with sql script to buil the backend üëç.

## Ussing Docker

Run `docker-compose build` and `docker-compose up`

## Development server

For local dev server run this commands:

in ms-cocktails:
`go run main.go`

in ms-users:
`go run main.go`
